<2017><2018>

也来写个总结吧，突然看到草稿箱里躺着的去年的总结，莫名的感慨。

一年过去，当初定下的目标完成了一部分。

2017已完成的

  1. 努力去考研了，虽然结果未知
  2. 改掉了许多坏毛病，如晚睡晚起。。不吃早饭等等
  3. 开始锻炼身体了

总之这一年成长了不少。

2018TODO

  1. 补完计算机基础，包括操作系统，数据库，组成原理，编译原理
  2. 学c， java （目前进度还行）
  3. 毕业论文
  4. 继续坚持锻炼

很多话不是很想说出来，就这样吧，希望明年再来看的时候，能觉得一年没有白费。

最后希望自己，能更坚强，更成熟，变得更强哈哈哈

加油2018！
